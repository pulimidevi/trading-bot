# Trading Bot (Binance Futures Testnet)

## Overview

This is a CLI-based Python trading bot that supports MARKET and LIMIT orders.

## Features

* BUY and SELL orders
* MARKET and LIMIT order types
* Command-line input
* Logging of requests and responses
* Error handling

## How to Run

Run the program:
python cli.py

## Example Input

BTCUSDT
BUY
MARKET
0.001

## Logs

All logs are stored in:
trading_bot.log

## Note

Orders are simulated due to API access restrictions, but the structure and response format match real Binance API behavior.
